# Beauty Spa Logo Instructions

## To Use the Tana's Beauty Boost Logo Image:

1. **Place your logo image file** in this directory: `/public/assets/images/`
2. **Name the file**: `Tana beauty boost.png` (or update the path in `BeautySpaLogo.jsx`)
3. **Supported formats**: PNG, JPG, JPEG, SVG, or WEBP

The component will automatically:
- Load the image if it exists
- Fall back to the SVG version if the image is not found
- Display properly in both header and hero sections

## Current Logo Path:
The component is looking for: `/assets/images/Tana%20beauty%20boost.png`

If your logo file has a different name or format, update the `logoSrc` variable in `src/components/BeautySpaLogo.jsx`
